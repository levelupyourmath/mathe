Maßeinheiten

Längen

  • $1\,\text{mm} = 0{,}1\,\text{cm}$
  • $1\,\text{cm} = 10\,\text{mm}$
  • $1\,\text{dm} = 10\,\text{cm}$
  • $1\,\text{m} = 10\,\text{dm}$
  • $1\,\text{km} = 1000\,\text{m}$

Merksatz: Von mm bis m wird es immer $10$-mal grösser, von m zu km $1000$-mal.

Flächen

  • $1\,\text{mm}^2 = 0{,}01\,\text{cm}^2$
  • $1\,\text{cm}^2 = 100\,\text{mm}^2$
  • $1\,\text{dm}^2 = 100\,\text{cm}^2$
  • $1\,\text{m}^2 = 100\,\text{dm}^2$
  • $1\,a = 100\,\text{m}^2$
  • $1\,\text{ha} = 100\,a$
  • $1\,\text{km}^2 = 10000\,\text{ha}$

Merksatz: Flächen werden jeweils $100$-mal grösser (Ausnahme bei ha und km²).

Volumen

  • $1\,\text{mm}^3$
  • $1\,\text{cm}^3$
  • $1\,\text{dm}^3$
  • $1\,\text{m}^3$
  • $1\,\text{km}^3$

Merksatz: Volumen wird immer $1000$-mal grösser.

Gewicht

  • $1\,\text{mg} = 0{,}001\,\text{g}$
  • $1\,\text{g} = 1000\,\text{mg}$
  • $1\,\text{kg} = 1000\,\text{g}$
  • $1\,\text{t} = 1000\,\text{kg}$
  • $1\,\text{kt} = 1000\,\text{t}$

Merksatz: Gewicht wird immer $1000$-mal grösser.

Zeit

  • $1\,\text{s}$
  • $1\,\text{min} = 60\,\text{s}$
  • $1\,\text{h} = 60\,\text{min}$
  • $1\,\text{d} = 24\,\text{h}$
  • $1\,\text{w} = 7\,\text{d}$
  • $1\,\text{a} = 52\,\text{w}$

Tipps

  • Länge → ×10
  • Fläche → ×100
  • Volumen → ×1000
  • Einheiten immer sauber hinschreiben
